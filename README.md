# BoilerPlate React App

## Install Express and Webpack
```
$ npm init
$ npm install express@4 --save
$ sudo npm install -g webpack@1.12.13
```

## install dependencies
```
$ npm install
```

## run webpack
```
$ webpack -w
# watch with -w
```

## run express
```
$ node server.js
```

And then head to http://localhost:3000
